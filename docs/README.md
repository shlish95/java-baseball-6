# ✏️ 구현할 기능 정리
## 1. 게임 준비
### ⬜️ 상대방(컴퓨터)이 숫자 생성 
### ⬜ 사용자가 숫자를 입력할 수 있는 환경 구현
## 2. 입력 받은 수 검증 
### ⬜ 3자리 수가 맞는지 확인
### ⬜ 다른 문자가 아닌, 숫자인지 확인
### ⬜ 서로 다른 수가 맞는지 확인
## 3. 입력 값에 대한 결과 출력
### ⬜ 볼 개수 확인
### ⬜ 스트라이크 개수 확인 
### ⬜ 결과 출력 
## 4. 게임 종료 후, 재시작 / 종료 구분
### ⬜ 종료 메시지 출력 및 값 입력 받기
### ⬜ 입력 값에 따라 재시작 / 종료 실행
